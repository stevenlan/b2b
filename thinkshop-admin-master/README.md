<h1 align="center" style="margin: 30px 0 30px; font-weight: bold;">ThinkShop v1.0.0</h1>
<h4 align="center">基于 Vue3/Element Plus 和 Spring Boot/Spring Cloud & Alibaba 前后端分离的分布式微服务架构</h4>


## 平台简介

* 本仓库为前端技术栈 [Vue3](https://v3.cn.vuejs.org) + [Element Plus](https://element-plus.org/zh-CN) + [Vite](https://cn.vitejs.dev) 版本。


## 前端运行

```bash
# 克隆项目
git clone https://gitee.com/zkthink/thinkshop-admin.git

# 进入项目目录
cd thinkshop-admin

# 安装依赖
yarn --registry=https://registry.npmmirror.com

# 启动服务
yarn dev

# 构建测试环境 yarn build:stage
# 构建生产环境 yarn build:prod
# 前端访问地址 http://localhost:80
```
