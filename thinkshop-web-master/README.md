<h1 align="center" style="margin: 30px 0 30px; font-weight: bold;">ThinkShop v1.0.0</h1>
<h4 align="center">基于 Vue3/Ant Design 和 Spring Boot/Spring Cloud & Alibaba 前后端分离的分布式微服务架构</h4>

## 平台简介

* 本仓库为前端技术栈 [Vue3](https://v3.cn.vuejs.org) + [Ant Design](https://antdv.com) + [Vite](https://cn.vitejs.dev) 版本。
* THINKSHOP 使用 AGPLv3 开源，请遵守 AGPLv3 的相关条款，或者联系作者获取商业授权

## 前端运行

```bash
# 克隆项目
git clone https://gitee.com/zkthink/thinkshop-web.git

# 进入项目目录
cd thinkshop-web

# 安装依赖
yarn --registry=https://registry.npmmirror.com

# 启动服务
yarn dev

# 构建生产环境 yarn build
# 前端访问地址 http://localhost:8086
```
